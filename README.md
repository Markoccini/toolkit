# Toolkit
Small Project for the sake of learning Java & Spring-Boot.
A website intended to have multiple subservices, currently only including the Backend of a Poll System

# Goals:
- [ ] Add Patch Route for a Poll where the Choices can be changed along with the Poll Question
- [ ] percentages for the votes of each Choice
- [ ] Build a Frontend using Vue.js, Vite & Tailwind CSS
- [ ] Communicating with an external API (Currency Exchange)
- [ ] Maybe a Login?

---
- [x] Proper Exception Handling


# Good to know
- After loading up the app and db, the swagger ui is accessible via
  `http://127.0.0.1:8080/swagger-ui/index.html` or `http://127.0.0.1:8080/docs` (redirect)
- The Frontend is accessible via default `http://127.0.0.1:5173`
